# EU-DRULE 🇪🇺

EU数字法规分析系统 - 专为NotebookLM设计的模块化提示系统，用于深度分析欧盟数字法规报告。

## 🎯 系统概述

EU-DRULE是一个多角色AI分析系统，专门用于解析和分析欧盟数字法规文件。系统采用模块化架构，包含核心引擎和可扩展插件包，为用户提供全面的法规分析能力。

### 核心功能
- **多角色分析**：法律专家、政策分析师、影响评估师、行动顾问、咨询专家、情报分析师
- **模块化扩展**：支持动态加载功能扩展包
- **结构化输出**：提供多种报告格式模板
- **元分析能力**：集成批判性思维和系统动力学方法

## 🔧 使用方法

### 基本使用
1. 将核心系统提示文件导入NotebookLM作为主要分析源
2. 根据需要加载相应的扩展包文件
3. 使用斜杠命令激活特定分析模式

### 扩展集成
- 使用NotebookLM的"Source"功能动态加载ADDON文件
- 不同扩展包可以组合使用以实现更复杂的分析
- 版本兼容性：核心v1.4支持所有ADDON v1.0扩展包

## 📊 版本信息

- **核心系统**: v1.3 → v1.4（增加ADDON支持）
- **扩展包**: v1.0（全部兼容）
- **语言支持**: 中文 + 英文版本
